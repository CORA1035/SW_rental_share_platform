# ShareRent - 물품 공유 및 대여 플랫폼 (Vanilla JS SPA)

주변 이웃과 안 쓰는 물품을 공유하고 대여할 수 있는 웹 서비스입니다.  
본 프로젝트는 **GitHub Pages**에 즉시 배포하여 실행할 수 있도록 백엔드 서버 없이 브라우저 클라이언트 사이드(**Vanilla HTML5, CSS3, ES6 Javascript**) 및 **`localStorage`** 데이터베이스 시뮬레이션으로 설계된 단일 페이지 애플리케이션(SPA)입니다.

## 🌟 주요 기능
1. **회원가입 & 로그인**: 실시간 형식 검사 및 중복 이메일 체크가 탑재된 로그인 세션 구현.
2. **물품 등록**: `FileReader API`를 활용한 사진 업로드(Base64 인코딩) 및 일일 대여 가격 설정.
3. **물품 검색 및 조회**: 키워드 검색, 카테고리 필터 칩, 최대 가격 필터, 정렬(최신순, 가격순) 기능 제공.
4. **대여 신청 및 관리**: 대여 기간 선택 시 비용 실시간 자동 계산 기능. 대시보드를 통해 대여 신청 수락/거절/반납 확인 처리.
5. **1:1 채팅**: 메시지 주고받기 및 대화 상대방의 답변을 유도하는 '자동 응답(Bot)' 기능 탑재.
6. **테스트 편의성 (계정 전환Dropdown)**: 헤더 영역에서 테스트 계정을 자유롭게 스위칭하여, 브라우저 1개의 창에서 빌리는 사람과 빌려주는 사람 역할을 완벽히 오가며 테스트 가능.

## 🚀 설치 및 시작하기 (로컬 실행)

이 프로젝트는 서버 환경이 필요하지 않으므로, 컴퓨터에 내려받아 `index.html` 파일을 더블 클릭하여 즉시 실행하거나 정적 서버(Live Server 등)를 통해 바로 실행할 수 있습니다.

### 1. 소스코드 복제
```bash
git clone <repository-url>
cd share-rental-platform
```

### 2. 실행
- 폴더 내의 **`index.html`** 파일을 웹 브라우저로 엽니다.
- 또는 VS Code 등 IDE의 `Live Server` 플러그인을 사용하여 실행할 수 있습니다.

## ☁️ GitHub Pages 배포 방법

1. GitHub 리포지토리를 새로 생성합니다. (예: `share-rental-platform`)
2. 리포지토리의 **Settings ➡️ Pages** 메뉴로 이동합니다.
3. **Build and deployment ➡️ Branch** 설정을 `main` 브라우저 및 `/ (root)` 폴더로 지정하고 **[Save]**를 클릭합니다.
4. 1~2분 후 제공되는 깃허브 전용 무료 호스팅 도메인(예: `https://<username>.github.io/share-rental-platform/`)으로 접속하여 정상 실행을 확인합니다.
